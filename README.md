# PDFHub

PDFHub is a browser-based PDF toolkit implemented as a static website. It provides a polished user interface and a comprehensive set of PDF utilities, all available without authentication, subscriptions, or backend infrastructure.

## Overview

This project is a front-end web application that serves as a central hub for PDF operations. It is optimized for ease of use in the browser and includes multiple utility pages for organizing, converting, editing, securing, and analyzing PDF documents.

## Key Features

- 32 distinct PDF tools
- Fully client-side experience
- No login or personal data required
- Modern responsive design
- Search-enabled tool catalog
- Static HTML/CSS implementation
- Author branding and social links included

## Repository Structure

- `index.html` — primary landing page and tool directory
- `tools/` — individual pages for each PDF utility
- `tools/shared.css` — shared styling for tool pages

## Usage

1. Open `index.html` in a web browser.
2. Select the desired PDF tool from the homepage.
3. Upload files and follow the tool-specific interface to process PDFs.
